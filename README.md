# wgx – Weltgewebe CLI

Eigenständiges CLI für Git-/Repo-Workflows (Termux, WSL, Linux, macOS).
Lizenz: MIT (projektintern).

Nutzung:
  ./wgx --help
  ./wgx version
  ./wgx selftest

Installation (Termux):
  git clone git@github.com:alexdermohr/wgx.git ~/.local/opt/wgx
  ln -sf ~/.local/opt/wgx/wgx ~/.local/bin/wgx
